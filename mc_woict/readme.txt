This widget enables a visitor to search for events located on the WhatsOnInCapeTown Web site.  (http://www.whatsonincapetown.com).


== Description ==

Many of the WhatsOnInCapeTown customers or visitors has asked for this widget to be developed.  
This widget enables a visitor to search for events located on the WhatsOnInCapeTown
Web site. The plugin installs a widget with a Date search control on a local site and then enables you to 
search for events on the WhatsOnInCapeTown website. (http://www.whatsonincapetown.com).

This Widget simply opens up a new browser window and  returns the search results from the WhatsOnInCapeTown database. This plugin can  be used on any WordPress site.


== Installation ==

1. Upload `mc_woict.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Widgets and drag and drop this widget to anywhere you like for it to show you on site. 

== Frequently asked questions ==

= Does it have an API? =

There is NO API as the plugin does not interact with the local server or website.
